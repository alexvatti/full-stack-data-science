
Step 1: Run the following command in the command prompt:
pip install -r requirements.txt

Step 2: Open the Jupyter Notebook (install Jupyter if not already installed).


Step 3: Clone the code from the GitHub repository:
https://github.com/alexvatti/full-stack-data-science/blob/main/Internship-Exercises/ML-Internship/Classification/Iris-Flower-Classification/Iris_Flowers_Classification.ipynb
https://github.com/alexvatti/full-stack-data-science/blob/main/Internship-Exercises/ML-Internship/Classification/Iris-Flower-Classification/iris_dataset.csv


Step 4: Open the cloned directory with Jupyter and run all cells. This will generate "iris-model.pkl".

Step 5: Open the Iris-Flowers-Prediction-App with VSCode.

Step 6:

1.Open the Anaconda command prompt.
2.Navigate to the directory of the Iris-Flowers-Prediction-App app:
cd <path_of_the_Iris-Flowers-Prediction-App>
3. Run the Streamlit app: streamlit run Iris-Flowers-Prediction-App.py

Step 7: A new webpage will be opened in the web browser.
 Enter the inputs  and click on the predict button.
